📊 Maven Market – Retail Sales & Performance Dashboard (Power BI)

This Power BI dashboard provides a comprehensive, end-to-end analysis of Maven Market’s retail performance.  
It helps business stakeholders monitor **profitability**, **sales trends**, **product performance**, and **regional contribution** through interactive and visually engaging insights.

The dashboard is designed with a **clean, professional layout**, featuring advanced Power BI capabilities such as DAX calculations, bookmarks, conditional formatting, and custom KPIs.

---

## 🎯 **Business Purpose**
The goal of this dashboard is to help decision-makers:

- Identify **top-performing product brands**
- Track **weekly revenue trends** and monthly KPIs
- Monitor **profit margins** and **return rates**
- Understand **geographical performance** across regions
- Compare **actual revenue vs target**
- Quickly filter insights using interactive regional buttons

---

## 📈 **Key Insights Provided**
- **Total Transactions:** Daily & monthly sales volume trends  
- **Total Profit:** Profit contribution by product brand  
- **Profit Margin %:** Product-level profitability comparison  
- **Return Rate:** Identify products with high customer returns  
- **Current Month KPIs:**  
  - Month-to-date transactions  
  - Month-to-date profit  
  - Month-to-date return count  
  Each KPI also shows goal vs actual performance.

- **Geo Analysis:**  
  - Map visualization showing customer concentration  
  - Tree map highlighting each country’s sales contribution  

- **Trend Analysis:**  
  Weekly revenue trending to observe performance patterns.

- **Revenue vs Target:**  
  Gauge chart comparing actual revenue vs sales target.

---

## 🛠 **Technical Highlights**
- Built using **Power BI Desktop**
- **DAX Measures** for dynamic KPIs  
- **Power Query** for data cleaning & transformation  
- **Conditional Formatting** for profit margin and return rate  
- **Bookmark Navigation** for advanced interactions  
- **Data Modeling** for optimized relationships  
- **Custom Buttons** for region-based filtering
